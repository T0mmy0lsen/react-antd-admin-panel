## Dev Notes

### CHANGELOG Format
```markdown
# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2025-12-XX

### Added
- TypeScript-first builder pattern API
- React 19 support
- Ant Design 6 integration
- Hooks API (useGet, usePost, useList, useForm, useAccess)
- CLI tool (create-raap-app)
- Storybook documentation

### Changed
- Complete rewrite from v1
- Modern ESM-first package

### Breaking Changes
- Requires React 19.1.0+
- Requires Ant Design 6.1.0+
- New API (not backwards compatible with v1)
```
